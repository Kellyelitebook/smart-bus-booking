# NairobiRide - Smart Bus Booking System

## Problem Statement
Travelers in Kenya waste time and money going to busy terminals. Bus prices change without notice, schedules are unreliable, and there is no convenient way to book from home or track buses.

## Solution
NairobiRide is a modern, beautiful, and user-friendly web app that lets users:
- Search routes across Kenya
- View real-time-like bus schedules
- See dynamic pricing
- Book tickets instantly from anywhere
- View live bus tracking (simulated)
- Manage their bookings

## Features
- Clean dark modern design
- Dynamic pricing simulation
- Live tracking visualization
- Responsive and beautiful UI
- Instant booking with confirmation

## How to Run
1. `pip install streamlit pandas`
2. `streamlit run app.py`
